# Savebox Landing Page
一頁式 app landing page，未使用 jQuery 或 CSS 框架。

## 下載並安裝相依套件
```bash
git clone https://github.com/mingjunlu/savebox-landing-page.git
cd savebox-landing-page
npm install
```

## 檢查程式語法錯誤
```bash
npm run lint
```

## 啟動開發伺服器
```bash
npm start
```

## 編譯、打包、輸出
```bash
npm run build
```
